About

TypeScript-based Model Context Protocol (MCP) server for processing CSV files and extracting organized URL links. Designed for integration with Claude Desktop.

Features

  • CSV file parsing from data directory
  • Automatic URL extraction from any column
  • Categorization and organization of URLs
  • MCP Tools API (3 tools available)
  • MCP Resources with csv:// URI scheme

Available MCP Tools

get_urls_from_csv
Extract and organize URLs from a specific CSV file
list_csv_files
List all available CSV files in the data directory
get_all_urls
Get all URLs from all CSV files, organized by filename

Technology Stack

TypeScript 5.7.2 Node.js 18+ MCP SDK 1.0.4 csv-parse 5.6.0 Vercel

Usage

This MCP server is designed to run locally with Claude Desktop via stdio transport. It is not a web API.

To use this server:

  1. Clone the repository from GitHub
  2. Run npm install and npm run build
  3. Configure Claude Desktop with the server path
  4. Use MCP tools in Claude conversations
