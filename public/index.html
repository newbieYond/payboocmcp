RESTful API for processing CSV files and extracting organized URL links. Built with TypeScript and deployed on Vercel serverless functions.

üöÄ Features

  • HTTP REST API endpoints
  • List available CSV files
  • Extract URLs from specific CSV files
  • Get all URLs from all files
  • Automatic categorization
  • CORS enabled
  • JSON responses

üîå API Endpoints

GET /api/csv-files

List all available CSV files in the data directory.

GET https://payboocmcp.vercel.app/api/csv-files

Response:

{
  "success": true,
  "count": 2,
  "files": [
    "example-links.csv",
    "sample-urls.csv"
  ]
}
üß™ Try it now

GET /api/extract-urls

Extract and organize URLs from a specific CSV file.

GET https://payboocmcp.vercel.app/api/extract-urls?filename=example-links.csv

Parameters:

  • filename (required) - Name of the CSV file

Response:

{
  "success": true,
  "filename": "example-links.csv",
  "data": {
    "urls": ["https://...", "https://..."],
    "categorized": {
      "Documentation": ["https://..."],
      "Tools": ["https://..."]
    },
    "total": 8
  }
}
üß™ Try it now

POST /api/extract-urls

Extract URLs using POST method with JSON body.

POST https://payboocmcp.vercel.app/api/extract-urls

Request Body:

{
  "filename": "example-links.csv"
}

Response:

{
  "success": true,
  "filename": "example-links.csv",
  "data": { ... }
}

GET /api/all-urls

Get all URLs from all CSV files, organized by filename.

GET https://payboocmcp.vercel.app/api/all-urls

Response:

{
  "success": true,
  "count": 2,
  "data": {
    "example-links.csv": {
      "urls": [...],
      "categorized": {...},
      "total": 8
    },
    "sample-urls.csv": {
      "urls": [...],
      "categorized": {...},
      "total": 4
    }
  }
}

üíª Usage Examples

JavaScript / Fetch API

// List CSV files
const response = await fetch('https://payboocmcp.vercel.app/api/csv-files');
const data = await response.json();
console.log(data.files);

// Extract URLs from specific file
const urls = await fetch(
  'https://payboocmcp.vercel.app/api/extract-urls?filename=example-links.csv'
);
const result = await urls.json();
console.log(result.data);

cURL

# List files
curl https://payboocmcp.vercel.app/api/csv-files

# Extract URLs (GET)
curl "https://payboocmcp.vercel.app/api/extract-urls?filename=example-links.csv"

# Extract URLs (POST)
curl -X POST https://payboocmcp.vercel.app/api/extract-urls \
  -H "Content-Type: application/json" \
  -d '{"filename":"example-links.csv"}'

Python

import requests

# List CSV files
response = requests.get('https://payboocmcp.vercel.app/api/csv-files')
files = response.json()['files']

# Extract URLs
response = requests.get(
    'https://payboocmcp.vercel.app/api/extract-urls',
    params={'filename': 'example-links.csv'}
)
data = response.json()['data']
